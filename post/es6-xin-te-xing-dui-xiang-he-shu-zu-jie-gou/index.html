ES6新特性 · 对象和数组解构

2020-08-03

ES6可以将对象中的属性或者数组中的元素进行解构，操作方式与前面所提到的Rest和Spread操作符类似，看一下下面这个栗子：

let person = {
    name: "Peter",
    age: 22,
    career: "student"
}

const {name, age, career } = person;
console.log(`Hello, my name is ${name}, and my career is ${career}.`);   
//Hello, my name is Peter, and my career is student.
