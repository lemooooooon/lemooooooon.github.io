几种左右固定宽度，中间自适应布局方法

2020-08-04

圣杯布局：

使用float实现

<style>
    * {
        margin: 0;
        padding: 0;
    }
    body {
        min-width: 550px;
    }
    .container > div {
        min-height: 200px;
    }
    .left {
        float: left;
        width: 200px;
        background: #ffbbff;
    }
    .middle {
        margin: 0 200px;
        background: #bfefff;
    }
    .right {
        float: right;
        width: 200px;
        background: #eeee00;
    }
</style>
</head>
<body>
    <div class="container">
    <div class="left">left</div>
    <div class="right">right</div>
    <div class="middle">middle</div>
    </div>
</body>
